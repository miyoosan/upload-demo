# upload-demo

* 并行模式可以在代码里修改worker数量，作者懒，就不做UI了。

* 本项目没有经过严格测试，也没有测试用例，感兴趣可以自己补足。

* 不少隐藏功能，自己探索吧，懒得写了。

* 如果跑起来访问不到页面，改一下devServer.public的对应端口
